    - 좌항 ?. 우항의 프로퍼티 (옵셔널 체이닝 연산자)
        
        
        | 좌항 | 반환값 |
        | --- | --- |
        | null 또는 undifined | undifined |
        | null , nudifined 이외의 것들 | 좌항.우항의프로퍼티 |
    - 좌항 ?? 우항 (null 병합 연산자)
        
        
        | 좌항 | 반환값 |
        | --- | --- |
        | null 또는 undifined | 우항 |
        | null , nudifined 이외의 것들 | 좌항 |
    
    ```jsx
    let elem = null
    
    let value = elem?.value
    console.log(value) //undefined
    
    let value = elem??value
    console.log(value) //null
    ```
